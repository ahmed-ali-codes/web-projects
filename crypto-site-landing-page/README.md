# CryptoSecure – 3D Crypto Investment Landing Page

[![Made with Three.js](https://img.shields.io/badge/Three.js-r128-green?logo=three.js)](https://threejs.org/)  
[![HTML5](https://img.shields.io/badge/HTML5-Valid-orange)](https://validator.w3.org/)  
[![CSS3](https://img.shields.io/badge/CSS3-Flex%2FGrid-blue)](https://www.w3.org/Style/CSS/)  
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**CryptoSecure** is a modern, responsive landing page for a fictional crypto investment platform. It features immersive 3D animations built with Three.js, a futuristic design, and smooth scroll navigation – ideal for showcasing a crypto startup or as a front‑end portfolio piece.

![Hero Screenshot](assets/images/screenshot-hero.png)

---

## ✨ Features

- **3D Particle Sphere** in the hero section (Three.js)
- **Floating Animated Cubes** in the investment section
- **Fully Responsive** layout (mobile, tablet, desktop)
- **Smooth Scroll** navigation with sticky header
- **Gradient & Glow Effects** (modern UI design)
- **WhatsApp Floating Button** (customizable contact link)
- **Performance Optimized** – lightweight Three.js scenes

---

## 🛠️ Tech Stack

| Area          | Technology                                      |
|---------------|-------------------------------------------------|
| Markup        | HTML5                                           |
| Styling       | CSS3 (Flexbox, Grid, backdrop‑filter)          |
| 3D Graphics   | Three.js r128 + OrbitControls                  |
| Icons         | Font Awesome 6.4.0                             |
| Fonts         | Google Fonts (Orbitron, Exo 2)                 |
| Animations    | CSS transitions, JavaScript requestAnimationFrame |
