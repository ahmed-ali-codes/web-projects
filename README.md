## 🚀 Getting Started

Each project is self-contained in its own folder. To run any project locally:

```bash
# Clone the repository
git clone https://github.com/ahmed-ali-codes/web-projects.git

# Navigate into the desired project
cd web-projects/coffies-site   # or any other project folder

# Open index.html in your browser
open index.html
```

> For projects with a PHP/MySQL backend, make sure you have a local server (e.g. XAMPP or Laravel Herd) running.
