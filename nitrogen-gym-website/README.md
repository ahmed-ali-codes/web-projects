## 📖 Description

**Nitrogen Gym** is a conceptual performance training facility. This website serves as its digital storefront, showcasing:

- Brand philosophy and mission
- Facility features (The Iron Room, High-O₂ Cardio, Sprint Track)
- Program details (Strength & Power, HIIT, Personal Training)
- Coach profiles (Strength, HIIT, Recovery, Bodybuilding)
- Class schedules with weekly tabs
- Membership tiers (Starter, Elite, Apex)
- Contact form and location information
- Legal pages (Privacy Policy, Terms of Service)

The site is fully responsive, includes a mobile burger menu, scroll reveals, and a language switcher that triggers **Google Translate** for automatic English ↔ Arabic translation.

---

## ✨ Features

- **Fully Responsive Design** – adapts to mobile, tablet, and desktop screens.
- **Bilingual Support** – one-click toggle between English and Arabic using Google Translate API.
- **Interactive Program Switcher** – dynamically updates program image and description.
- **Class Schedule Table** – includes weekly view and special class highlighting.
- **FAQ Accordion** – collapsible questions on the Membership page.
- **WhatsApp Floating Button** – direct chat link for inquiries.
- **Instagram Social Link** – integrated in navbar and footer.
- **Scroll Reveal Animations** – smooth fade‑in as elements come into view.
- **Fixed Navigation** – with background transparency change on scroll.
- **Modern Visual Aesthetics** – dark theme, red accents, custom fonts (Rajdhani & Montserrat).
