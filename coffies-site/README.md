## 📖 Project Description

**Coffies** is a fully responsive, front‑end e‑commerce website for a premium coffee brand. It features a rich user interface with a hero section, product catalogue, interactive cart drawer, smooth animations, and a multi‑step checkout. All cart data persists via `localStorage`, and an integrated mock API simulates order placement – ideal for demonstrating modern front‑end development skills.

## ✨ Features

- 🛒 **Dynamic shopping cart** – add/update/remove items with real‑time total calculation
- 💾 **Persistent storage** – cart contents saved in `localStorage`
- 📦 **Smart shipping logic** – free shipping for orders over $40
- 🎨 **Immersive UI** – full‑screen hero, scroll animations, custom cursor, shimmer effects
- 📱 **Fully responsive** – mobile, tablet, and desktop layouts
- 🔍 **Menu filtering** – filter products by category (espresso, latte, cold brew, beans, gift sets)
- 🖼️ **Image gallery** – masonry‑like grid with hover zooms
- 📝 **Contact form** – client‑side validation (no backend – demo ready)
- 💳 **Checkout flow** – form validation, order summary, mock API call, and success page
- 🍞 **Toast notifications** – user feedback for cart actions
- 🧭 **Smooth navigation** – sticky navbar, anchor scrolling, active page highlighting

---

## 🛠️ Tech Stack

- **Front‑end:** HTML5, CSS3 (Flexbox, Grid, custom properties, animations)
- **JavaScript (ES6+):** DOM manipulation, modules, classes, async/await
- **Storage:** `localStorage` for cart persistence
- **Fonts:** Google Fonts (Playfair Display, Inter)
- **Icons:** Emoji / Unicode (no external icon libs)
- **Mock API:** Custom `api.js` simulating backend order processing
- **Deployment Ready:** Static files – works on any web server (Netlify, Vercel, GitHub Pages)
